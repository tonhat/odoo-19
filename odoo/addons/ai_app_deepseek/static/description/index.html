AI Ask DeepSeek

This module extends the AI App in Odoo 19 Enterprise Edition to add support for the DeepSeek AI model. With this module, users can use their own DeepSeek account for the "Ask AI" feature in Odoo.

Key Features

  • DeepSeek Model Integration: Adds the deepseek-chat model as a new language model option.
  • Custom API Key: Allows users to configure their own DeepSeek API key in the Odoo settings.
  • Seamless Experience: Easy to install and configure.

Setup and Configuration

After installing the module, navigate to Settings -> General Settings -> AI, where you will find the option to add your DeepSeek API key.
